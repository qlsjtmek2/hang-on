## 시나리오 #1: 감정 털어놓기

**사용자**: 힘든 감정을 느끼는 20대 여성 (지은)
**목표**: 자신의 감정을 기록하고 공유하여 위로받기
**맥락**: 밤 11시, 침대에서 혼자 있을 때, 힘든 하루를 보낸 후

### 단계

1. **앱 실행**
   - 스플래시 화면 (< 3초)
   - 메인 화면 진입

2. **털어놓기 시작**
   - 화면 하단의 "털어놓기" 플로팅 버튼 탭
   - 감정 선택 화면으로 전환

3. **감정 상태 선택**
   - 5개 날씨 아이콘 중 "비 🌧️" 선택
   - 선택된 아이콘 확대 + 색상 강조 효과
   - 자동으로 글쓰기 화면 전환

4. **글 작성**
   - 텍스트 입력: "오늘 하루가 너무 힘들었어요. 모든 게 잘 안 풀리는 것 같아요..."
   - 실시간 글자 수 표시: "137/500"
   - "다 썼어요! 💙" 버튼 탭

5. **공유 설정**
   - 바텀시트 표시: 3가지 옵션
   - "지금 나누기" 선택
   - 확인 메시지: "마음을 전달했어요 💙"

6. **완료**
   - 메인 화면으로 돌아감
   - 작성한 기록이 목록 최상단에 표시됨

### 감정 변화

- **시작**: 우울함 (2/5), 외로움, 답답함
- **중간**: 표현을 통한 카타르시스, 약간의 해방감
- **종료**: 약간의 위안 (3/5), 누군가 봐줄 거라는 기대감

### 주요 Touchpoint

- 메인 화면 (기록 목록)
- 감정 선택 화면
- 글쓰기 화면
- 공유 옵션 바텀시트

### 성공 지표

- 감정 선택 완료율 > 95%
- 글 작성 완료율 > 80%
- 공유 설정 완료율 > 90%

## 시나리오 #2: 누군가와 함께 보기

**사용자**: 공감을 주고 싶은 30대 남성 (민준)
**목표**: 비슷한 감정을 느끼는 사람의 기록을 읽고 공감 전달
**맥락**: 점심 시간, 잠깐의 휴식 중

### 단계

1. **앱 실행**
   - 메인 화면 진입
   - 자동으로 자신의 최근 기록 기반 감정 분석 (백그라운드)

2. **누군가와 함께 탭**
   - 하단 탭바에서 "누군가와 함께" 탭 선택
   - 감정 유사도 매칭된 기록 피드 로딩 (< 2초)

3. **피드 조회**
   - 헤더에 "오늘 남은 이야기: 20/20" 표시
   - 카드 형식으로 기록 표시:
     - 날씨 아이콘 (예: 🌧️)
     - "3시간 전"
     - 글 내용 미리보기 (2줄)
     - 공감 수, 메시지 수
   - 스크롤하며 여러 기록 탐색

4. **기록 상세 보기**
   - 흥미로운 기록 카드 탭
   - 전체 내용 표시
   - 하단에 "공감하기 💙" 및 "메시지 보내기" 버튼

5. **공감 전송**
   - "공감하기 💙" 버튼 탭
   - 하트 펄스 애니메이션 (300ms)
   - 공감 수 +1 업데이트
   - 버튼 비활성화 (이미 공감함)

6. **조회 수 차감**
   - 헤더 업데이트: "오늘 남은 이야기: 19/20"
   - 피드로 돌아감

### 감정 변화

- **시작**: 호기심, 여유로움
- **중간**: 공감, 연민, "나도 그랬어" 공감
- **종료**: 뿌듯함, 연결감

### 주요 Touchpoint

- 메인 화면
- 누군가와 함께 탭 (피드)
- 기록 상세 화면
- 공감/메시지 전송 버튼

### 성공 지표

- 평균 조회 기록 수: 15-20개/일
- 공감 전송률 > 30%
- 메시지 전송률 > 10%

## 시나리오 #3: 공감 및 위로 받기

**사용자**: 위로가 필요한 사람 (수진)
**목표**: 자신의 기록에 달린 공감과 메시지 확인
**맥락**: 아침, 기록 작성 후 하루가 지난 시점

### 단계

1. **푸시 알림 수신**
   - 잠금 화면에 알림 표시
   - "누군가 당신의 마음에 공감했어요 💙"
   - 탭하여 앱 실행

2. **메인 화면 진입**
   - 어제 작성한 기록 카드에 뱃지 표시
   - "하트 +5, 메시지 +2" 표시

3. **기록 상세 보기**
   - 기록 카드 탭
   - 전체 글 내용 표시
   - 스크롤 다운하여 받은 공감/메시지 확인

4. **공감 확인**
   - "공감 5개" 섹션
   - 하트 아이콘 5개 표시 (익명)
   - 각 하트에 애니메이션 효과

5. **메시지 확인**
   - "받은 메시지 2개" 섹션
   - 메시지 카드 표시:
     - "힘내세요 💪"
     - "저도 그래요 🫂"
   - 익명 표시 (작성자 정보 없음)

6. **감정 변화 인지**
   - 따뜻함, 위로받는 느낌
   - "혼자가 아니구나" 깨달음

### 감정 변화

- **시작**: 궁금함, 설렘
- **중간**: 감동, 따뜻함
- **종료**: 위로받음 (4/5), 감사함, 연결감

### 주요 Touchpoint

- 푸시 알림
- 메인 화면 (뱃지 표시)
- 기록 상세 화면
- 공감/메시지 목록

### 성공 지표

- 푸시 알림 오픈율 > 40%
- 공감/메시지 확인율 > 95%
- 앱 재방문율 증가

## 시나리오 #4: 내 기록 관리

**사용자**: 기록을 정리하고 싶은 사람 (현우)
**목표**: 과거 기록 조회, 수정, 삭제
**맥락**: 주말, 한가한 시간

### 단계

1. **메인 화면 진입**
   - 자신의 기록 목록 표시 (시간순 정렬)
   - 각 카드: 날씨, 시간, 글 미리보기, 공감 수, 메시지 수

2. **기록 필터링 (선택)**
   - 감정별 필터: "비 🌧️만 보기"
   - 공개 상태별 필터: "비공개만 보기"

3. **기록 수정**
   - 특정 기록 카드 탭
   - 우상단 "..." 메뉴 → "수정하기"
   - 글쓰기 화면 진입 (기존 내용 표시)
   - 내용 수정 후 "다 썼어요! 💙" 탭
   - 공유 설정 유지 또는 변경
   - "수정 완료" 확인 메시지

4. **기록 삭제**
   - 다른 기록 카드 탭
   - 우상단 "..." 메뉴 → "삭제하기"
   - 확인 다이얼로그: "이 기록을 삭제할까요? 받은 공감과 메시지도 함께 사라져요."
   - "삭제" 버튼 탭
   - 목록에서 즉시 제거
   - 하단 스낵바: "삭제되었어요" (30일 복구 가능 안내)

5. **공개 설정 변경**
   - 비공개 기록 카드 탭
   - "지금 나누기"로 변경
   - 확인 다이얼로그: "이 기록을 공개로 변경할까요?"
   - "변경" 탭
   - 감정 유사도 매칭 풀에 추가됨

### 감정 변화

- **시작**: 정리 욕구, 차분함
- **중간**: 과거 감정 회상, 감정 정리
- **종료**: 깔끔함, 만족감

### 주요 Touchpoint

- 메인 화면 (기록 목록)
- 기록 상세 화면
- 수정 화면
- 삭제 확인 다이얼로그
- 공개 설정 변경 다이얼로그

### 성공 지표

- 기록 수정률 < 10% (낮을수록 좋음 - 첫 작성에서 만족)
- 기록 삭제율 < 5%
- 공개 설정 변경률 > 15%

## 시나리오 #5: 첫 사용 온보딩

**사용자**: 처음 앱을 설치한 신규 사용자 (서연)
**목표**: 앱의 목적과 사용법 이해, 첫 기록 작성
**맥락**: 앱 스토어에서 설치 직후

### 단계

1. **앱 첫 실행**
   - 스플래시 화면
   - 온보딩 화면 진입 (3-4 슬라이드)

2. **온보딩 슬라이드**
   - **슬라이드 1**: "감정을 날씨로 표현해요 ☁️"
     - 5단계 날씨 아이콘 소개
   - **슬라이드 2**: "완전히 익명으로 공유해요 🎭"
     - 익명성 보장 설명
   - **슬라이드 3**: "비슷한 감정의 사람들과 연결돼요 🫂"
     - 감정 유사도 매칭 설명
   - **슬라이드 4**: "하루 20개까지 읽을 수 있어요 📚"
     - 일일 제한 설명 (과몰입 방지)

3. **회원가입**
   - "시작하기" 버튼 탭
   - 회원가입 화면 진입
   - 이메일/비밀번호 입력 또는 Google 로그인
   - 약관 동의 체크박스
   - "가입하기" 탭

4. **첫 기록 작성 안내**
   - 메인 화면 진입 (빈 상태)
   - Empty State 일러스트: "아직 기록이 없어요"
   - "첫 번째 감정 털어놓기" CTA 버튼
   - 버튼 탭 → 감정 선택 화면

5. **첫 기록 작성**
   - 날씨 선택: "구름 조금 🌤️" (긍정적 시작)
   - 글 작성: "처음 써보는 감정 일기예요. 새로운 시작이 기대돼요!"
   - "다 썼어요! 💙" 탭
   - 공유 옵션: "혼자 간직하기" (안전하게 시작)

6. **완료 및 다음 단계 안내**
   - 메인 화면에 첫 기록 표시
   - 툴팁: "누군가와 함께 탭에서 다른 사람들의 이야기를 읽어보세요 👉"
   - 온보딩 완료

### 감정 변화

- **시작**: 호기심, 낯설음
- **중간**: 이해, 흥미
- **종료**: 기대감, 준비됨

### 주요 Touchpoint

- 온보딩 슬라이드
- 회원가입 화면
- Empty State (빈 메인 화면)
- 첫 기록 작성 Flow
- 완료 툴팁

### 성공 지표

- 온보딩 완료율 > 70%
- 첫 기록 작성률 > 60%
- 첫 방문 후 24시간 내 재방문율 > 40%
