## 레이아웃 설계 원칙

### 1. 8pt Grid System

모든 요소는 8의 배수로 정렬합니다:

- **간격**: 8, 16, 24, 32, 40, 48pt
- **컴포넌트 높이**: 44, 48, 56pt (터치 영역 최소 44pt)
- **화면 여백**: 좌우 16pt

### 2. Visual Hierarchy

- **제목**: 가장 큰 폰트 (28pt), 상단 배치
- **주요 콘텐츠**: 중간 폰트 (16pt), 중앙 배치
- **보조 정보**: 작은 폰트 (14pt), 회색 텍스트

### 3. Safe Area

- **상단**: 상태바 + 노치 고려
- **하단**: 홈 제스처 영역 고려 (iPhone X 이상)
- React Native: `SafeAreaView` 사용

### 4. 터치 영역

- **최소 크기**: 44x44pt (WCAG 2.1 AA)
- **권장 크기**: 48x48pt
- **버튼**: 최소 높이 48pt

### 디자인 상세

**Header**:

- 높이: 56pt
- 좌측: 타이틀 "Hang On" (h2, 24pt, Bold)
- 우측: 설정 아이콘 (24x24pt)
- 배경: `colors.background` (#FFFFFF)
- 하단 구분선: 1pt, `colors.border`

**Record Card**:

- 높이: 최소 120pt (내용에 따라 가변)
- 패딩: 16pt (상하좌우)
- 간격: 카드 간 16pt
- 배경: `colors.surface` (#F5F5F5)
- 모서리: 12pt border-radius
- 그림자: elevation 2 (Android) / shadowOpacity 0.1 (iOS)

**Floating Button**:

- 크기: 56x56pt
- 위치: 우하단 (우측 16pt, 하단 탭바 위 16pt)
- 배경: `colors.primary` (#4A90E2)
- 아이콘: 흰색 "+" (24x24pt)
- 그림자: elevation 4

**Tab Bar**:

- 높이: 56pt
- 3개 탭: 동일한 너비
- 선택된 탭: `colors.primary` 색상 + Bold
- 비선택 탭: `colors.text.secondary` (#757575)

### 디자인 상세

**Emotion Button**:

- 크기: 80x80pt (기본), 96x96pt (선택됨)

**인터랙션**:

- 탭하면 확대 + 배경색 표시
- 0.5초 후 자동으로 글쓰기 화면 전환

## 컴포넌트 명세

### 재사용 가능한 컴포넌트

#### 1. RecordCard

**용도**: 기록 목록 카드 (내 기록, 피드)

**Props**:

- `emotion`: 감정 레벨 (1-5)
- `time`: 작성 시간
- `content`: 글 내용 (미리보기)
- `empathyCount`: 공감 수
- `messageCount`: 메시지 수
- `onPress`: 탭 이벤트

**레이아웃**:

```
┌─ RecordCard (120pt) ──────────────────┐
│ [emotion] [emotionText]    [time]     │  ← Header
│                                        │
│ [content preview - 2 lines]            │  ← Content
│                                        │
│ 💙 [empathyCount]  💬 [messageCount]  │  ← Footer
└────────────────────────────────────────┘
```

#### 2. Button

**용도**: 모든 버튼

**Variants**:

- `primary`: 파란색 배경 (CTA)
- `secondary`: 흰색 배경 + 테두리
- `text`: 배경 없음 (텍스트만)
- `destructive`: 빨간색 (삭제)

**States**:

- `default`: 기본 상태
- `pressed`: 탭 시 (10% 어둡게)
- `disabled`: 비활성화 (50% 투명도)

#### 3. EmotionButton

**용도**: 감정 선택 버튼

**Props**:

- `level`: 감정 레벨 (1-5)
- `selected`: 선택 여부
- `onPress`: 탭 이벤트

**레이아웃**:

```
┌─ EmotionButton (80x80pt) ─┐
│                            │
│       [emotion icon]       │  ← 48pt
│       [emotion text]       │  ← 16pt
│                            │
└────────────────────────────┘
```

#### 4. BottomSheet

**용도**: 하단 시트 (공유 옵션, 메시지 프리셋)

**Props**:

- `visible`: 표시 여부
- `children`: 내용
- `onClose`: 닫기 이벤트

**애니메이션**:

- 진입: translateY (아래 → 위, 300ms)
- 퇴장: translateY (위 → 아래, 300ms)

#### 5. Dialog

**용도**: 확인 다이얼로그

**Props**:

- `visible`: 표시 여부
- `title`: 제목
- `message`: 메시지
- `buttons`: 버튼 배열
- `onClose`: 닫기 이벤트

**레이아웃**:

```
┌────────────────────────────┐
│                            │
│  [title - h3]              │
│                            │
│  [message - body]          │
│                            │
│  ┌──────────┐ ┌──────────┐│
│  │  취소    │ │   확인   ││  ← 2개 버튼
│  └──────────┘ └──────────┘│
└────────────────────────────┘
```
